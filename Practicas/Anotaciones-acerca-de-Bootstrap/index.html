Anotaciones sobre Bootstrap

Nav

  • La clase .navbar solo aplica a la etiqueta <nav>.
  • Utilizar la clase .navbar-expand-sm para hacer que cuando el ancho sea menor de 576px.
  • La etiqueta <ul> debe tener la clase .navbar-nav.
  • Ir hacia aquipara tener un nav responsive y alineado
  • La clase .justify-content se ele aplica al elemento padre de la etiqueta <ul>

    • Flex

    • La clase .d-flex y las sucesivas clases en el contenedor
    • Usa las clases .align-* para cambiar el alineamiento de los elementos a (solo funciona en inline, inline-block, inline-table y table cell elementos)
